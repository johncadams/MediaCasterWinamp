Getting Started

Media Caster extends Winamp's Media Library integrating into the tree.  Upon connecting to a Media Caster server the index of songs are downloaded and displayed in the main window which then can be searched against or played in Winamp's main window.

Getting started

  1. Download the latest release (listed above)
  2. Install Media Caster (winamp will also be installed if you don't already have a copy)
  3. Start Winamp
  4. Open Winamp's Media Library window (press Alt-L)
  5. Select Media Caster in the tree (left side)
  6. Enter your username/password (first time use-only)
  7. The index should start downloading, once complete select a song and press "Play"

Media Caster integrates into Winamp's Media Library (pressing Alt-L will bring up this window).  Selecting Media Caster in the tree will connect to the server downloading the song and play lists into the list area.  From the list area you can browse, search, or play the songs.  The status area at the bottom indictaes how many songs are displayed in the list.

When a song is played it will appear in Winamp's playlist window (not shown, press Alt-E) in the play order.  From this window one can re-arrange the order or remove songs.  For more information on Winamp consult its documentation.



Tree
 select Media Caster to begin downloading song and playlists.
List
 list of songs in the Media Caster library (or playlist).  The list may be sorted by selecting one of the column headings.
Search
 searchs for songs containing any of the entered words (in any field).  Use quotemarks for exact matches.
Play
 plays the selected song(s).
Enqueue
 enqueues the selected song(s).
Refresh
 downloads the song and play lists.
Config
 brings up the configuration dialog.



Host
 server name
Port
 port number
User
 username for account (if required)
Password
 password for account (if required)
Enabled
 disables Media Caster (too confusing, removed in v1.0)
Auto-update
 checks for software updates
Update
 check for updates on demand
Empty
 removes all content from main window
Refresh
 downloads the song and play lists
Bitrate
 controls the audio quality (lower quality = better network performance)

    NOTE:  Changing the bitrate only effects new songs selected to be played, not the ones listed in the Playlist editor.

Frequently Asked Questions

Can I download MP3's from Media Caster?  
No, Media Caster uses MP3 streaming and no files are ever transported.

Is Media Caster free? 
Yes and always will be (unless someone pays me tons of money... or sues me).

Is Media Caster secure? 
Yes, Media Caster depends upon HTTP-based security

Can I operate my own Media Caster server?.
Up until now I have worked primarily on the client with the server-side a bit more complicated.  In the next phase of developement I plan to improve the server code and make it available for Linux installations.

How does it really work?

What else can I do?
